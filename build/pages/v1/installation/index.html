Installation

The Alpheios Version 1 tools are deprecated. You must be using Firefox Version 45 or earlier to install them.

Please install the tools from their official home on the Firefox Add-Ons Site.

Installation from Firefox Add-Ons Site:

  1. Click the link to go to the Firefox Add-Ons Site
  2. Once on the site, click the green "Add to Firefox" buttons to install the Alpheios Greek Tools and Alpheios Latin Tools.
  3. You must install the Alpheios Basic Libraries as well as at least one of the Alpheios Language Tool Extensions (Latin and/or Greek and/or Arabic ).
  4. Wait for the Firefox Software Installation dialog to appear (this may take a few seconds):
  5. Click "Install Now" to install the extensions.
  6. Restart Firefox
