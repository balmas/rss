Getting Started

After installing the Alpheios Reading Tools extension for Chrome or Firefox, the Alpheios icon will be added to your browser's toolbar. Click on it to Activate Alpheios.

Alternatively, you can activate Alpheios from the browser's context menu.

The Alpheios panel will open. This panel contains instructions for accessing the word lookup functionality that Alpheios adds to the browser pages on which you activate it.

If you close this panel, you can always reopen it via the Info entry in the Alpheios Reading Tools submenu in browser's context menu or via Options button in the Alpheios popup.
